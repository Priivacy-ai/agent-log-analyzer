Allowed Tool Recommendations

Agent Analyzer recommends only public, URL-backed tools from this allowlist. Private MCP names, private skills, raw paths, prompts, and transcript text are not used as recommendation labels.

Free Report Candidates

  • ccusage - Claude Code usage accounting and burn-rate visibility.
  • ccstatusline - statusline telemetry for context/cost awareness.
  • Context Mode - context compression, telemetry, and output hygiene.
  • Claude Code Hooks Mastery - vetted hook patterns for session hygiene.
  • claude-context - MCP semantic retrieval for large repositories.
  • claude-token-efficient - response terseness guidance for smaller future context.

Review-Only / Research Candidates

These may appear in internal analysis or paid-pack research, but should not be presented as a one-click install unless the registry marks them allowed and the source URL matches the recommendation.

  • awesome-claude-code - ecosystem index for monitoring new tools.
  • Claude Code Usage Monitor - burn-rate monitoring and forecasting.
  • Claude Code Usage Tracker - lightweight usage telemetry.

Explicit Exclusions

We do not currently recommend npm packages or GitHub repos whose public identity does not match the claimed token-saving behavior. For example, the public npm package named rtk is not a shell-output reducer, so it must not be recommended as one.
